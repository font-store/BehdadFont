## Install

#### Linux
gnu/inux users could use [kateb](https://github.com/kiamazi/kateb) font manager, to install Behdad font.

ubuntu users must install this dependensies too:
```
sudo apt install openssl libssl-dev
````

##### install kateb
```
sudo cpan kateb

kateb install behdad
or
kateb update behdad
```
